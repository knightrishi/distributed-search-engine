**Distributed Search Engine**

A distributed search engine built from scratch using Spring Boot, MySQL, and Jsoup.

A distributed search engine built from scratch using Spring Boot, MySQL, and Jsoup.

Instead of relying on Elasticsearch or Solr, this project focuses on implementing the core components of a search engine manually: web crawling, indexing, ranking, sharding, and distributed query processing.

**Tech Stack**
Java 21+
Spring Boot
Spring Data JPA
Hibernate
MySQL
Maven
Jsoup
